# Cognitive Complexity & complexipy

Presentation slides for the Python Cali Meetup about cognitive complexity and the [complexipy](https://github.com/rohaquinlop/complexipy) library.

## Getting Started

```bash
bun install
bun run dev
```

Open [localhost:3030](http://localhost:3030) to start editing.

## Scripts

| Command | Description |
|---|---|
| `bun run dev` | Start dev server |
| `bun run build` | Build static site |
| `bun run export` | Export to PDF |
